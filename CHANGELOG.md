# 1.1

- Added High Contrast support
- Updated to 1.21.11
- Pressing ESC on the config screen now discards any unsaved options, just like the cancel button does
- Added tooltips to the page edit buttons
- Tooltips now appear when hovering labels in the config screen instead of just the widget
- Added the option to prefer vanilla book screens per book type, useful if you have other mods that add to the vanilla book screen
- Keybind inputs can now be unbound by pressing escape
